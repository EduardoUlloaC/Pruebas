# xdt-Shop API

1. Clonar proyecto
2. Instalar librerias:

```
npm install
```

3. clonar el archivo `.env.template` y renombrar a `.env`
4. Cambiar las variables de entorno
5. Levantar la base de datos:

```
docker-compose up -d
```

6. Ejecutar SEED

```
http://localhost:3000/api/seed
```

7. Levantar:

```
npm run start:dev
```
